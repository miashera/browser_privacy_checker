About CVSS Scoring

The Common Vulnerability Scoring System (CVSS) is a framework for communicating the characteristics and severity of software vulnerabilities. This extension uses CVSS to rate privacy risks.

Severity CVSS Score Range Description
Critical 9.0 - 10.0 Critical privacy risks that require immediate attention
High 7.0 - 8.9 High-risk privacy issues that should be addressed quickly
Medium 4.0 - 6.9 Moderate privacy concerns that should be reviewed
Low 0.1 - 3.9 Low-risk privacy issues that have minimal impact

Report History

The extension keeps a history of the last 10 privacy reports. You can view and export these reports from the extension popup.
